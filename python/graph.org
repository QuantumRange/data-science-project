Next, we need to create a graph from the data. For this we use
[[https://networkit.github.io/][networkit]]. The graph will have
- a node for every crawled website that
  is entry in the database,
- and an directed edge for every link from one website to anoter website
  in the database.
But since the data is not necessarily complete, we need to make sure
to ignore all links to urls outside of the database.

Since nodes are represented as integers internally, we have to figure
out a mapping from urls to ints. The index of df would be the easiest
answer, but figuring out the index of a url is linear in time. Thus we
will first sort the database index by the urls lexicographically,
which will in turn enable us to find an index in logarithmic
time. This requires [[https://numpy.org/][numpy]].

#+begin_src python :results none :noweb-ref imports :tangle no
  import numpy as np
#+end_src

#+begin_src python :results none
  # Create unique sorted url index for fast lookups.
  df.sort_values("url", inplace=True)
  df.reset_index(inplace=True)
  df.drop('index', axis=1, inplace=True)
  index = np.unique(df.url.to_numpy())

  G = nk.Graph(len(index), directed=True)
#+end_src

Now we can finally also add our edges. We also assign weights to our edges:
This happens by aggregating all our links from the same domain into one edge
weight. Since this is also just a snapshot of our data all links that are not
found in the index will be ignored.

#+begin_src python :noweb-ref imports :tangle no
  from collections import Counter
#+end_src

#+begin_src python
  # Aggregate data points with the same url and create a corresponding weighted
  # edge in the graph.
  for i, url in enumerate(index):
      il = df.url.searchsorted(url, side="left")
      ir = df.url.searchsorted(url, side="right")
      
      c = Counter()
      for idx in range(il, ir):
          for l in df.links.iloc[idx]:
              c[l] += 1
      c = dict(c)

      # Calculate edge weights
      for k,v in c.items():
          w = v / (ir - il)
          j = index.searchsorted(k)
          if j >= 0 and j < len(index) and index[j] == k:
              G.addEdge(i, j, w)
#+end_src
